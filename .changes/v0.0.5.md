## v0.0.5 - 2025-12-09
### Added (Patch, new features)
* added jwk signing key endpoints, jwk, user & group repo methods
* Added auth middleware support to the API server. This includes:
- API key management to represent machine identities.
- Oauth2 & OIDC authentication methods for users with cookie and header based authentication.
- STS token exchange for enabling upstream applications to exchange user tokens for internal tokens.
- JWT signing key management for signing and verifying JWTs.

* added initial sdk-y build
